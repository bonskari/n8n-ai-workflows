Quick Setup

  1. Install n8n and Ollama
  2. Pull a model: ollama pull llama3:8b
  3. In n8n: Settings → Import from File → select the JSON
  4. Update the model name in HTTP Request nodes if using a different model
  5. Configure any credentials (IMAP for email, etc.)
  6. Activate the workflow
