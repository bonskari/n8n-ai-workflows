## Requirements

- [n8n](https://n8n.io) self-hosted (v1.0+)
- [Ollama](https://ollama.ai) running on localhost:11434
- Any Ollama model (`llama3:8b` recommended)

## Quick Start

1. Download a template JSON
2. In n8n: **Settings → Import from File** → select the JSON
3. Update the model name if using something other than `llama3:8b`
4. Configure any required credentials (IMAP for email, etc.)
5. Activate the workflow
